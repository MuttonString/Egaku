## 如何运行前端应用程序

```shell
cd front-end
npm install
npm run dev
```

## 如何运行后端应用程序

```shell
cd front-back
# Windows:
.\venv\Scripts\actuvate
# macOS/Linux:
source venv/bin/activate
pip install -r requirements.txt
python src/main.py
```
