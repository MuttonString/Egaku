## フロントエンドアプリの実行方法

```shell
cd front-end
npm install
npm run dev
```

## バックエンドアプリの実行方法

```shell
cd front-back
# Windows:
.\venv\Scripts\actuvate
# macOS/Linux:
source venv/bin/activate
pip install -r requirements.txt
python src/main.py
```
